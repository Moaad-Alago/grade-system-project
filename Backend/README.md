# Grade System (Full-Stack) - Web Development Project

A full-stack grade management system built with **HTML, CSS, JavaScript** for the frontend, and **Node.js** + **Express** for the backend,
with **Redis Cloud** as the database.

## Features

- Add a student grade for a selected course
- Automatic final grade calculation based on course components
- Pass / Fail status calculation
- View all students and their courses
- Search student by ID
- Filter by status (**Passed / Failed**)
- Edit a student's course grade
- Delete a course for a student
- Delete a student
- Data persistence using **Redis Cloud**

-----

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript 

### Backend
- Node.js
- Express.js
- CORS
- dotenv
- redis (Node Redis client)

### Database
- Redis Cloud
